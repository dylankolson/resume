Ticket Tracker

This was an internal tool I created for PhocusDesigns. It allows to mange projects, tasks and employees. Admins can create projects and assign tasks to each project. Tasks then can be assigned to a certain employee. Information is stored about each ticket, such as date submitted, description, status and date completed. It replaced Google Documents as a way to mange tasks within the company.

Penn State Police Scanner

Here is a working website I created for a friend, it currently has just over 100 users. The website allows people to create an account and sign up for text and email alerts for police events around campus. Admins who monitor the Police Scanner then can create an event and tag it with a location and details. If you are signed up for the tagged location you will receive the alert. Check it out here: PSUScanner.com

Awesome Apartments

Awesome Apartments was a group project in my database design class. This site was to allow users to search for apartments based on certain criteria. Users can also post their own properties to the website, which includes uploading a picture. Here is a video by a group member demoing our website: Awesome Apartments

Cravings Application

Cravings was a project in my Intermediate Java Class. The basic idea of the app was to be able to find a restaurant serving a certain food. This was a proof of concept, where a list of 8000 foods were imported and serialized. Restaurants and their menus were created at random for the purpose of this project. Users can search for a food and be directed to the restaurants info page for more info.
